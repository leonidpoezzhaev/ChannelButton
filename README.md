<h1 align="center">ChannelButton</h1>
<h4 align="center">Bot for publishing posts with buttons</h4>

## Идея
Бот для публикации постов вместе с кнопкой по каналам.

Возможность публикации постов сразу в несколько каналов (хоть во все).

Частичная реализация платной подписки.

## Смысл
Бот для практики в уже изучаемых навыках.

"Набивание руки" и экспериментирование с Sqlite3 и Telebot
